# Как использовать
1) Положить скрипт в папку с распакованным архивом и Excel-таблицей
2) Установить библиотеки (если не установлены):

```bash
pip install pandas openpyxl pyyaml
```
3) Изменить настройки в начале файла (пути к папкам, имя Excel-файла)
4) Запустить скрипт:

```bash
python pipeline.py
```
